CSS Grid Property Reference

Property Description Islamic Analogy
grid-template-columns/rows Defines track sizes Like the pillars and arches defining a mosque's structure
grid-template-areas Visual layout template Like the precise geometric patterns in Islamic art
gap (gutter) Space between tracks Like the breathing space between arabesque designs
grid-column/row Item placement Like the mihrab indicating prayer direction
justify/align-items Alignment within cells Like the symmetry in a perfectly aligned prayer row
fr units Fractional space distribution Like the fair division prescribed in Islamic inheritance

Grid vs Flexbox

Flexbox is for 1D layouts (like a straight prayer line)

Grid is for 2D layouts (like complex mosque floorplans)

Fr Units

Fractional units distribute space like inheritance in Islam:

grid-template-columns: 1fr 2fr

→ Second column gets twice the space of the first

Naming Grid Lines

Like naming mosque gates for orientation:

grid-template-columns:
[main-start] 1fr [content-start] 1fr [content-end] 1fr [main-end];
