CSS Positioning Reference

Position Value Behavior Islamic Analogy
static (default) In normal document flow Like a worshipper in straight prayer rows
relative Positioned relative to itself Like adjusting your position within your prayer space
absolute Positioned relative to nearest positioned ancestor Like the mihrab indicating qibla direction
fixed Positioned relative to viewport Like the Kaaba - fixed reference for all
sticky Hybrid of relative and fixed Like a prayer leader who stays visible when others scroll

Key Positioning Concepts

  • Document Flow: Like the natural order of prayer rows (static positioning)
  • Positioning Context: The ancestor that becomes the reference point (like the mosque structure)
  • Z-Index: The stacking order, like priority in prayer lines
  • Fixed vs Absolute: Fixed to viewport (like the Kaaba) vs absolute to parent (like mihrab in a mosque)
