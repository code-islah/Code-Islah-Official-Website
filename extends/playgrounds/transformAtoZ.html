Property Reference

Property Description Common Values
transform Applies 2D/3D transformations translate(), rotate(), scale(), skew(), matrix()
transform-origin Sets the origin point for transformations center, top left, 50% 50%, 20px 20px
transition Animates property changes property duration timing-function delay
transition-timing-function Controls animation progression ease, linear, ease-in, ease-out, cubic-bezier(), steps()
background Gradient backgrounds linear-gradient(), radial-gradient(), conic-gradient()

Timing Function Cheat Sheet

Linear

Constant speed: linear

Ease (Default)

Slow start/end, fast middle: ease

Ease-in

Slow start: ease-in

Ease-out

Slow end: ease-out

Ease-in-out

Slow start/end: ease-in-out

Steps

Discrete jumps: steps(5)
