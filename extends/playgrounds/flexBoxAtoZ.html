Flexbox Property Reference

Property Applies To Description Values
flex-direction Container Main axis direction row | row-reverse | column | column-reverse
flex-wrap Container Single or multi-line nowrap | wrap | wrap-reverse
justify-content Container Alignment along main axis flex-start | flex-end | center | space-between | space-around | space-evenly
align-items Container Alignment along cross axis (per line) stretch | flex-start | flex-end | center | baseline
align-content Container Alignment when extra space in cross axis stretch | flex-start | flex-end | center | space-between | space-around
gap Container Space between items length values (px, em, rem, etc.)
order Items Visual reordering integer (negative/positive)
flex-grow Items Ability to grow if space available unitless number (default 0)
flex-shrink Items Ability to shrink if needed unitless number (default 1)
flex-basis Items Initial size before growing/shrinking length | auto | content
align-self Items Override container's align-items auto | flex-start | flex-end | center | baseline | stretch

Main Axis

Defined by flex-direction:

  • row: Left to right
  • row-reverse: Right to left
  • column: Top to bottom
  • column-reverse: Bottom to top

Cross Axis

Perpendicular to main axis:

  • For row: Vertical
  • For column: Horizontal

Shorthand

flex: grow shrink basis

  • flex: 1 = 1 1 0
  • flex: auto = 1 1 auto
  • flex: none = 0 0 auto
